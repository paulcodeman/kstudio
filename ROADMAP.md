## Roadmap — Node Graph для событий CMM

### Phase 1 — Фундамент

- [ ] Выбрать библиотеку: Rete.js или кастомный Canvas/SVG
- [ ] Подключить граф к панели событий (вкладка вместо textarea)
- [ ] Базовая нода **Event** (click, keydown, timer…)

### Phase 2 — Базовые ноды CMM

- [ ] `DrawBar` / `DrawText` / `DrawCaptButton`
- [ ] `SetText` / `Show` / `Hide` — управление виджетами
- [ ] `Timer` — `@SetTimer`
- [ ] `Condition` — if/else
- [ ] `Delay` — пауза

### Phase 3 — Интеграция

- [ ] Каждое событие открывает воркспейс графа вместо редактора кода
- [ ] Старый код импортируется как текстовая нода `[Raw CMM]`
- [ ] Кнопка «Посмотреть код» — генерация плоского CMM из графа
- [ ] Сохранение графа в атрибут компонента (JSON)

### Phase 4 — Продвинутое

- [ ] Copy/paste нод между событиями
- [ ] undo/redo графа
- [ ] Кастомные ноды — пользователь пишет CMM-шаблон → нода
- [ ] AI-генерация графа по описанию
